# FNP-Sales-Analysis-Dashboard
📊FNP (Ferns and Petals) For Gifts Sales Analysis Dashboard - Excel Project

I recently developed a Sales Analysis Dashboard in Excel to explore sales data and uncover meaningful business insights.

📈 What the dashboard highlights:

• Revenue and overall sales performance

• Number of orders and transaction trends

• Sales patterns over time

• Best-selling products and categories

• Customer buying behavior

This dashboard allows businesses to:

• Keep track of performance in a simple, visual way

• Spot trends and opportunities 

• Make smarter, data-driven decisions

✅ Tools & Techniques:

Microsoft Excel (Pivot Tables, Power Query, Interactive Charts, Slicers, Dashboard Design)

✅ Converting raw data into clear insights that support better decision-making.
